##  URDF 说明
URDF（Unified Robot Description Format）是 ROS 机器人描述文件，本仓库中的 URDF 文件定义了天工行者机器人的：
- 机械结构
- 关节限位
- 质量分布
- 惯性参数

该文件适用于 ROS 及 Gazebo 仿真，可用于运动规划与控制算法研究。
